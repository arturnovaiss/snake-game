# 🐍 Snake Game

Um clássico jogo da cobrinha desenvolvido com HTML, CSS e JavaScript puro.

## 🎮 Sobre o Projeto

O Snake Game é uma recriação do tradicional jogo da cobrinha, onde o jogador controla uma serpente que cresce ao consumir alimentos espalhados pelo mapa. O objetivo é alcançar a maior pontuação possível sem colidir com as paredes ou com o próprio corpo.

Este projeto foi desenvolvido com foco em lógica de programação, manipulação do DOM e desenvolvimento de jogos utilizando JavaScript Vanilla.

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* Canvas API

## ✨ Funcionalidades

* Controle da cobrinha por teclado
* Sistema de pontuação
* Crescimento progressivo da cobra
* Detecção de colisão
* Tela de Game Over
* Reinício da partida
* Interface responsiva e intuitiva
